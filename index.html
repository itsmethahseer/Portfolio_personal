Recent Projects

My recent projects span Generative AI, featuring a RAG Chatbot and an Invoice Document Processing system. I also have a strong background in Machine Learning, demonstrated through projects such as Pneumonia Detection, Weather Prediction, and ID Card Extraction.

Whizz Chatbot

Developed a document based chat application using Llama2, OpenAI, fastAPI, Langchain, Faiss, whisper and ChromaDB.Implemented RAG Prompting for getting answers when we ask a question that is in the uploaded dataset.

Invoice Parser

It is a application which can extract Information in a structured format such as JSON. Implemented Prompt Engineering with the help of multiple models such as Gpt-3.5 turbo , Gpt4-o , Gpt4-o-mini and Gemini 1.5 Pro. Deployed the entire application using FastAPI and Docker. Also used several technologies like Pdfplumber, AWS OCR extract, Chain of thought prompting, etc.

ID Card Extractor

Developed an API that can extract specific information from a National ID. Used Tesseract to extract the text from images. Used Gliner NER model to detect entity information and finetuned it. Annotated the samples using LabelImg tool for training with YOLO V5.

AOF Validation

This application will verify number of Handwritten fields in the Application Form meets the required number of fields. If they match and after verifying they are correct it will Accept the Form else will reject it. Utilized YOLO , Tesseract and FastAPI for deployment.

Pneumonia Detection

Developed a accurate machine learning model for detecting a patient having Pneumonia or not by calculating probablilty. Conducted some preprocessing on images and used pre trained model Resnet15 using Transfer learning. Created a API for testing the model using FastAPI.

Weather Prediction

Developed a accurate machine learning model for Predicting the temperature in specific location using humidity , pressure etc. Conducted feature selection methods , Preprocessing techniques and model selection. Used Linear regression , Decision tree , Random Forest , Support vector Regressor, for creating the model . Saved the predictive model using pickle library.
